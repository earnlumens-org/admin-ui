# Admin UI

EarnLumens administrative frontend built with Vue 3, Vite, and Vuetify.

## Requirements

- Node.js 20+
- npm 10+

## Local Development

```bash
npm install
npm run dev
```

## Useful Scripts

```bash
npm run build
npm run preview
npm run type-check
npm run lint
```

## Base Structure

- `src/` main application
- `src/pages/` views and pages
- `src/components/` reusable components
- `src/router/` routing
- `src/stores/` global state
